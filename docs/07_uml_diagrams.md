## Sequence Diagrams

### Spell Check Flow
1. User types word
2. Frontend sends CHECK request
3. Backend validates via Trie
4. Result returned to frontend

### Dictionary Lookup Flow
1. User searches word
2. Frontend sends DEFINE request
3. Backend searches Hash Table
4. Definition returned
