# User Guide

## TypeMaster – Intelligent Text Engine

---

## Introduction

This user guide explains how to use each module of **TypeMaster – Intelligent Text Engine**.
The application is designed to be intuitive and requires no prior configuration once it is running.

All features are accessible through the sidebar navigation menu.

---

## Application Dashboard

After launching the application, the main window displays:

* Sidebar navigation menu
* Active working panel
* Connection status indicator

The green indicator confirms that the frontend is successfully connected to the backend server.

---

## Typing Master

### Purpose

The Typing Master module is designed to improve typing accuracy and speed through controlled typing exercises.

### How to Use

1. Select **Typing Master** from the sidebar
2. Choose a typing sample from the dropdown list
3. Click inside the input area
4. Start typing the displayed text

### Rules

* Text must be typed exactly as shown
* Backspace key is disabled
* Incorrect characters are not allowed
* Exercise ends automatically on completion

### Metrics Displayed

* Time elapsed
* Completion progress
* Words Per Minute (WPM)

---

## Performance Dashboard

### Purpose

This module provides real-time typing performance analysis.

### How to Use

1. Open **Performance Dashboard** from the sidebar
2. Type freely in the text area
3. Observe live performance updates

### Metrics Explained

* **WPM**: Typing speed based on standard calculation
* **CPM**: Characters typed per minute
* **Accuracy**: Percentage of correct words
* **Total Words**: Total number of words typed
* **Invalid Words**: Misspelled words detected

---

## Dictionary Explorer

### Purpose

Allows users to search for word meanings from the built-in English dictionary.

### How to Use

1. Select **Dictionary Explorer** from the sidebar
2. Enter a word in the search field
3. Press Enter or click the search button

### Features

* Multiple definitions per word
* Part-of-speech categorization
* Fast lookup using hash table

---

## AI Word Predictor

### Purpose

Provides intelligent word suggestions for misspelled or incomplete words.

### How to Use

1. Open **AI Word Predictor**
2. Enter a word or misspelling
3. Press Enter or click Predict

### Output

* Ranked list of suggested words
* Confidence-based ordering
* Top matches highlighted

The prediction system uses **Edit Distance** and **MaxHeap ranking**.

---

## Live Spell Checker

### Purpose

Performs real-time spell checking while typing.

### How to Use

1. Open **Live Spell Checker**
2. Type text in the editor area
3. Misspelled words are underlined in red

### Correction

* Click on an underlined word
* Select a suggested correction
* Word is replaced automatically

---

## Print Report

### Purpose

Generates a consolidated report of user performance across all modules.

### How to Use

1. Select **Print Report** from the sidebar
2. Review the displayed statistics
3. Click the Print button

### Report Includes

* Typing Master performance
* Dashboard statistics
* Spell checking results
* Dictionary and prediction usage

---

## Connection Status

* Green indicator: Connected
* Red indicator: Disconnected

If disconnected, ensure the backend server is running.

---

## Conclusion

The user interface of TypeMaster is designed to be simple, efficient, and informative.
Each module serves a distinct purpose while working together to deliver a complete typing and text analysis experience.
